    • Usando radio/checkbox/select

    O truque é atribuir variáveis aos seletores, então mudar o valor das variávels quando determinado checkbox/select seja ativado, em casos de muitos checks como essa página atribuir um #id para cada section ou div pode ajudar. Os checkboxs tem mais prioridade do que select&options

    Preferencialmente é bom manter tudo no mesmo block e assim chamar o element pai por tag/class/id, se chamar section:has e o block estiver fora não vai funcionar, desse modo vai ser necessário chamar por body:has ou html:has assim não importa onde estiver vai funcionar, no caso de radio/checkbox especificar usando [type="radio"][id="html"].
