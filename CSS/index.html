Multi class:

Usando apenas a tag p

                  p.center {
                    text-align: center;
                    color: red;
                  }                  
                  p.large {
                    font-size: 300%;
                  }

A classe center foi definida em conjunto com a tag p então outras tags não serão afetadas declarando a classe em questão <h1>não foi definida

                  <h1 class="center">This heading will not be affected</h1>
                  <p class="center">This paragraph will be red and center-aligned.</p>
                  <p class="center large">This paragraph will be red, center-aligned, and in a large font-size.
                  </p>
