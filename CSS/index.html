Multi class: apenas a tag p
                  p.center {
                    text-align: center;
                    color: red;
                  }                  
                  p.large {
                    font-size: 300%;
                  }

A classe center e large foi especifica para a tag p e não global, então outras tags não serão afetadas mesmo declarando a classe em questão <h1> não será afetado. 

                  <h1 class="center">This heading will not be affected</h1>
                  <p class="center">This paragraph will be red and center-aligned.</p>
                  <p class="center large">This paragraph will be red, center-aligned, and in a large font-size.
                  </p>
